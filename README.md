# 🎬 Filmorate - java

**Filmorate** — Проектная деятельность. Это бэкенд-приложение для социальной сети, где пользователи могут оценивать фильмы, добавлять друзей и получать рекомендации.

Проект демонстрирует навыки построения REST API, работы с базами данных, написания тестов и следования принципам ООП и SOLID.

## 📌 Что реализовано

### 🎥 Работа с фильмами
- Создание, чтение, обновление фильмов
- Добавление и удаление лайков
- Получение топа популярных фильмов по количеству лайков

### 👥 Работа с пользователями
- Регистрация и обновление данных пользователей
- Добавление в друзья (односторонняя дружба)
- Получение списка друзей и общих друзей

### 🏷 Справочная информация
- Получение списка всех жанров (`GET /genres`)
- Получение списка всех рейтингов MPA (`GET /mpa`)
- Получение конкретного жанра или рейтинга по ID

## 🔗 API Эндпоинты

### Фильмы
- `GET /films` — все фильмы
- `GET /films/{id}` — фильм по ID
- `POST /films` — создать фильм
- `PUT /films` — обновить фильм
- `PUT /films/{id}/like/{userId}` — поставить лайк
- `DELETE /films/{id}/like/{userId}` — убрать лайк
- `GET /films/popular?count={n}` — топ фильмов

### Пользователи
- `GET /users` — все пользователи
- `GET /users/{id}` — пользователь по ID
- `POST /users` — создать пользователя
- `PUT /users` — обновить пользователя
- `PUT /users/{id}/friends/{friendId}` — добавить в друзья
- `DELETE /users/{id}/friends/{friendId}` — удалить из друзей
- `GET /users/{id}/friends` — друзья пользователя
- `GET /users/{id}/friends/common/{otherId}` —общие друзья


### Справочники
- `GET /genres` — все жанры
- `GET /genres/{id}` — жанр по ID
- `GET /mpa` — все рейтинги
- `GET /mpa/{id}` — рейтинг по ID

## 💾 Работа с базой данных

Одна из ключевых задач проекта — **миграция с in-memory хранения на полноценную базу данных**.

### Что сделано:
- Спроектирована реляционная схема из 7 таблиц:
    - `films` — информация о фильмах
    - `users` — пользователи
    - `genres` — справочник жанров
    - `mpa_ratings` — справочник рейтингов
    - `film_genres` — связь фильмов с жанрами
    - `likes` — лайки пользователей
    - `friends` — связи дружбы
- Настроены внешние ключи для обеспечения целостности данных
- Написаны SQL-запросы с JOIN для получения связанных данных
- Реализованы DAO-классы с использованием JdbcTemplate и RowMapper
- Настроена H2 Database в двух режимах:
    - Файловый режим (`jdbc:h2:file:./db/filmorate`) — для сохранения данных между запусками
    - In-memory режим — для интеграционных тестов


### Пример запроса:
```java
// Получение фильма с жанрами и рейтингом
String sql = "SELECT f.*, m.name as mpa_name " +
             "FROM films f " +
             "LEFT JOIN mpa_ratings m ON f.mpa_id = m.id " +
             "WHERE f.id = ?";
```
